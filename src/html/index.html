Usage guideline

Snacks@console is a PHP based application. Before you use it, you have to make sure that your system is ready to run PHP applications. All web server such as Laragon (recommended) or XAMPP provide the requirement to run a PHP based application.

After you setup your webserver. Open the command line inside the web server's root directory, usally www, and follow along the installation process

Using composer

$ composer create-project mazentouati/snacks-console "console"

Extracting package ( not recommended )

Alternatively, you can just extract the console file inside your webserver's root directory. First you need to download the package then extract it and you are ready to use the console.
